A simple URL shortener API built with Python and FastAPI, using MySQL for data persistence.

**FEATURES** 
- Create short URLs from valid links, using a hash (MD5)
- Automatic redirection when accessing a short URL
- URL validation on input
- Persistent storage using MySQL
- MySQL isolated using Docker
